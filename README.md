# GioroX AI, Inc. — Website

This repository contains the static landing page for **GioroX AI, Inc.**, the company building **WAP3** — a Web3-based agentic payment and provenance protocol.

WAP3 enables AI agents to:

- Escrow funds before they start working
- Orchestrate multi-step tasks across multiple models and tools
- Attach verifiable proofs and receipts to each run
- Settle payments on-chain once conditions are satisfied

The site is built as a single static HTML page using Tailwind CSS via CDN, and can be hosted on GitHub Pages or any static file host.

## Tech stack

- Static HTML
- Tailwind CSS via CDN
- No build step, no framework required

## Deployment (GitHub Pages)

1. Push this repository to GitHub.
2. In the repo, go to **Settings → Pages**.
3. Under **Build and deployment**, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Save. GitHub will publish the site at:

   `https://<your-username>.github.io/gioroxai-site`

You can then point a custom domain (for example `gioroxai.com` or `dreamxai.com`) to this URL using URL forwarding or DNS records at your domain registrar.
